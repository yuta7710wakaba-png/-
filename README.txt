療育教材「みんなでみつけよう！」 Ver1.0

【起動方法】
index.htmlを開いてください。

【収録教材】
・なにがかわった？
・難易度：★☆☆☆☆
・ねらい：共同注意／視覚探索／少し前を思い出す

【操作方法】
・右半分を押す：次へ
・左半分を押す：戻る
・右上の×：教材を終了
・画面長押し：先生用メニュー

【ファイル構成】
・index.html：画面構成
・style.css：デザイン
・script.js：操作と表示処理
・data.js：教材データ
・images：今後の画像追加用

【今後の追加】
data.jsに教材データを追加し、imagesフォルダに画像を入れる形で拡張できます。

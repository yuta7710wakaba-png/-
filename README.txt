・公園編　難易度★1　10問
・お祭り編　難易度★2　10問
・プール編　難易度★2　10問

【難易度の考え方】
★1：大きな物の出現・消失、分かりやすい色の変化
★2：数、種類、服や小物の変化
★3：背景の小物、位置、数など視覚探索が必要な変化

【操作方法】
・メニューから編を選択
・最初の絵をよく見る
・右側を押すと次の絵へ
・左側を押すと前の絵へ
・画面長押しで先生用メニュー
・PCとスマートフォンのメニューは縦スクロール対応

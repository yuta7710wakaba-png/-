療育教材「みんなでみつけよう！」 Ver2.1（フラット構成）

【変更理由】
Ver2.0ではscenesフォルダとquestionsフォルダが必要でした。
GitHubへフォルダごとアップロードできなかった場合、問題ファイルが読み込まれず、
「表示できる問題がありません」と表示されます。

Ver2.1では、すべてのファイルを同じ階層へ配置しました。

【アップロード方法】
ZIPを解凍し、中にあるファイルをすべてGitHubリポジトリの最上位へアップロードしてください。
既存ファイルがある場合は上書きしてください。

【主なファイル】
index.html
style.css
script.js
scene-loader.js
question-loader.js
scene-park.js
scene-festival.js
q001.js～q030.js

【収録問題】
公園：7問
お祭り：3問
合計：10問

【注意】
ファイル名は変更しないでください。
特にq001.js～q030.js、scene-park.js、scene-festival.jsが不足すると、
対応する問題や風景が表示されません。

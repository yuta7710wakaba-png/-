療育教材「みんなでみつけよう！」 Ver1.1

【概要】
大型モニターで職員が操作する、電子紙芝居形式の療育教材です。
Ver1.1では、ページをめくるような自然な画面切り替えを追加しました。

【起動方法】
index.html を開いてください。

【操作】
・画面右半分：次のページへ
・画面左半分：前のページへ
・右上の×：教材を閉じる
・画面長押し：先生用メニュー

【Ver1.1の変更点】
・右へ進むと、紙面が右から左へめくれる演出
・左へ戻ると、逆方向へめくれる演出
・切り替えの途中に白い紙面を表示
・連続タップによる二重切り替えを防止

【構成】
index.html / style.css / script.js / data.js / images

今後はdata.jsと画像を追加することで教材を増やせる構成です。

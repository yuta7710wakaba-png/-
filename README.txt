【使い方】
1. index.htmlを開きます。
2. 「はじめる」を押します。
3. 画面右半分で次へ、左半分で戻ります。
4. 右上の×で終了します。
5. 画面を約1秒長押しすると先生用メニューが開きます。

GitHub Pagesへ更新する場合は、このフォルダ内のファイルを既存リポジトリへ上書きしてください。
